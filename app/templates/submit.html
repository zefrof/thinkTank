Submit An Event

Instructions
  • Copy URL from an event's decks page.
  • Paste URL into submission box.
  • Click "Submit"
  • Our site will collect whatever info it can, then display it for editing.*
  • Edit the information and submit it for review.
Additional Info
  • * CURRENTLY ONLY COLLECTS INFO FROM MTGO LINKS FROM: https://magic.wizards.com. All other events have to be entered manually by us (for now).
  • Gathering deck information from the submitted link can take a while. Please exercise patience.
  • If you don't know the Number of Players, put the number of decks.
  • If it's a 5-0/4-1/etc dump (or the like) put the decks record for finish.
  • If the deck (sub)archetype isn't available, leave it un-selected. We'll add the correct one.
  • Cards are entered in this format: Quantity Card Name *Card Id.
    • 3 Lightning Bolt *32a180dd-9641-4546-8940-9b21084f0c71
    • 2 Delver of Secrets // Insectile Aberration
  • Card id's will be autofilled if not entered.
  • Card Names must be exact. EX: "Lightning Bolt" not "lightning bolt"
  • Everything except deck (sub)archetype is required.
